# Membuat Blank List

Untuk membuat Lists dari kosong / empty Lists, buka aplikasi Lists dari Microsoft 365 Launchpad

![HomeMicrosoft365](/gettingstarted/launchpad.png)

Klik pada bagian **New list** untuk membuat List baru

![Microsoft Lists](/gettingstarted/listshomepage.png)

Untuk membuat **blank list**, klik pada button **Blank List**

![Microsoft Lists](/gettingstarted/blanklist.png)

Akan muncul pop-up untuk form input terkait nama, deskripsi, dan icon dari List yang akan dibuat, isikan sesuai kebutuhan anda

![Microsoft Lists](/gettingstarted/popupformblanklist.png)

Selanjutnya, akan direct ke List yang dibuat, akan muncul tampilan List hanya dengan **Title** column dan untuk column lainnya dapat ditambahkan kemudian

![Microsoft Lift](/gettingstarted/listapp.png)
